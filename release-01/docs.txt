Files:

flag_tool.c:
A tool for inspecting all tweets made by users, by querying the databse.
It is possible with this tool to 'flag' tweets by feeding the program with the ids of tweets to flag.

minitwit.py:
The web application itself, 
containing a number of action methods for rendering 
different parts of the webpage

control.sh:
A bash script for performing different actions of the web application,
such as cleanup, startup and shutdown.
